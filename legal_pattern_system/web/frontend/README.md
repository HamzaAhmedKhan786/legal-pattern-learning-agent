# Frontend Scaffold

Recommended production frontend: **React + TypeScript + Vite**.

Why this stack:

- fast local development,
- strong TypeScript support,
- easy integration with FastAPI/OpenAPI-generated clients,
- suitable for upload, review, redline, and QA dashboards.

Suggested production screens:

- document upload,
- template review,
- case-data intake,
- generated draft viewer,
- QA/trace inspector,
- lawyer approval/redline feedback.
